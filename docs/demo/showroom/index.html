How to read PRE / POST / DELTA

  • PRE — observation window before a workflow adjustment; describes the friction pattern.
  • POST — observation window after an agreed adjustment; describes changed evidence patterns.
  • DELTA — qualitative comparison between PRE and POST; what appears improved and what remains open.

Read each scenario in order: PRE → POST → DELTA. Language is qualitative only — no numeric diagnostics.

# Tier 3 Enriched Handoff Readout

> This is a synthetic public-safe showroom example. It is not generated by the public Action Preview and does not expose the private SDVM engine, scoring internals, calibration, thresholds, weights, statistical methodology, or playbook logic.

## Purpose

This scenario illustrates what SDVM can communicate when **enriched handoff, decision, and trace evidence** is available — a Tier 3-style public-safe evidence profile. It contrasts with the Tier 2 examples (`handoff_ambiguity`, `reviewer_loop`, `late_checkpoint`), which surface friction candidates without mechanism-level corroboration.

Scenario identity: **SHOWROOM-T3-HANDOFF / public-safe synthetic derivative**

## How this differs from Tier 2 examples

| Tier 2 examples | This Tier 3 example |
|---|---|
| Surface operational friction candidates | Connect friction to a more specific mechanism |
| Candidate attribution remains open | Candidate mechanism corroborated within scenario |
| Limited enriched fields in evidence | Handoff summaries, owner/receiver, decision status, reason codes available |
| Structures pilot questions | Supports stronger mechanism-level attribution readout |

Tier 2 examples show that SDVM can surface friction and structure pilot questions. This example shows that **enriched evidence can strengthen attribution** — without claiming absolute proof or production diagnosis.

## PRE

PRE enriched traces show that handoff summaries and decision-status fields were missing or incomplete at Draft → Review transfer points. Downstream clarification was associated with missing context rather than only with normal review movement.

Observable pattern: repeated clarification after handoff points, with thin handoff summaries and implicit owner/receiver roles.

## POST

POST enriched traces show clearer handoff summaries, explicit owner/receiver metadata, and fewer clarification reason codes after targeted handoff repairs were applied.

The improvement is observed within this PRE/POST scenario. Enriched fields make the mechanism more observable than in Tier 2 readouts.

## DELTA

The DELTA readout supports an observed reduction in handoff friction and stronger mechanism-level attribution. The candidate mechanism — incomplete context transfer at handoff — is corroborated within this scenario, while broader generalization still requires additional comparable traces.

## Attribution boundary

Tier 3-style enriched evidence can strengthen mechanism-level attribution by connecting observed friction to specific missing or repaired handoff evidence. It still does not prove a sole root cause. Broader attribution requires additional comparable traces and deployment-specific review.

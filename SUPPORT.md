# Support

## Status

SDVM Action Preview is a **public-safe preview slice** — a composite GitHub Action (`validate-only`, `preview-report`, `synthetic` modes), local CLI, and documentation. It is not the full SDVM engine and is not a commercially mature or enterprise-supported product.

## How to get help

Use GitHub Issues with the structured templates under [`.github/ISSUE_TEMPLATE/`](.github/ISSUE_TEMPLATE/):

| Template | Use for |
|----------|---------|
| **GitHub Action bug report** | Action step failures, missing artifacts, workflow integration problems |
| **Documentation feedback** | Incorrect, missing, or unclear documentation |
| **Report quality feedback** | Report clarity, recommendations, confidence wording |
| **Adapter request** | Future observability source requests (no implementation commitment) |
| **Product preview feedback** | General adoption experience and preview suggestions |

For security concerns, see [`SECURITY.md`](SECURITY.md).

Do **not** include secrets, credentials, private traces, customer data, or sensitive payloads in public issues.

## Support expectations

This preview includes **best-effort** community support via GitHub Issues only.

It does **not** include:

- an enterprise SLA or guaranteed response time;
- a dedicated support inbox or paid support contract;
- production incident response guarantees.

## Marketplace status

GitHub Marketplace publication remains **on hold** pending explicit operator approval.
